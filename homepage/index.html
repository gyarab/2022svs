Vítejte na studentském vývojovém serveru AVAVA

Server je k dispozici zákům Gymnázia Arabská pro vzdělávací účely. Studenti si tu mohou například hostovat stránky svého ročníkového projektu, nebo je protunelovat a zabezpečit pomocí SSH a automatického HTTPS.

Jak se mohu registrovat?

Tvorba účtů je automatická a hlídána přes Google OAuth. Na server se lze zaregistrovat jedině přes studentský email končící na @student.gyarab.cz. Pro vytvoření nového účtu se připojte přes SSH na: 

$ ssh hello@svs.gyarab.cz

Nechci TOFU! Aby nebylo potřeba TOFU, můžete SSH host klíč stáhnout zde a dát si ho do known_hosts souboru. Získání klíče serveru touto metodou je bezpečnejší, ale není potřeba.

Na serveru logy nešetříme, prosím neprovádějte žádné neplechy :)
